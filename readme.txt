To commit/push changes:

(from inside the cs482team directory)

git status (to view staged/unstaged changes)

git add * (to stage all changes) (alternatively can add individual files)

git status (to confirm that changes were staged (will show up in green))

git commit -m "COMMIT MESSAGE" (write a useful commit message)

git push origin master (to push to the online repo)
